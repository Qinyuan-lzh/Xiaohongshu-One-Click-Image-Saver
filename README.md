# 小红书图片一键保存

一个尽量简单的小红书 Userscript。

它只做一件事：在小红书图文笔记详情页右侧显示一个 `保存图片` 按钮，点击后把当前笔记里的全部图片一次性保存下来。

## 功能

- 在图文笔记详情页右侧显示固定按钮
- 一键保存当前笔记的全部图片
- 支持将图片保存为 `jpg`、`png` 或保留原始格式
- 支持配置相对浏览器默认下载目录的子目录
- 不包含悬浮菜单、设置面板、说明弹窗等额外界面

## 安装方法

1. 安装一个 Userscript 管理器，例如 Tampermonkey。
2. 将 [`XHS-Downloader.js`](./XHS-Downloader.js) 导入到 Userscript 管理器。
3. 打开小红书图文笔记详情页，确认右侧出现 `保存图片` 按钮。

## 使用方法

1. 打开任意一篇小红书图文笔记详情页。
2. 点击右侧的 `保存图片` 按钮。
3. 脚本会自动保存当前笔记中的全部图片。

说明：

- 视频笔记不会显示按钮。
- 如果当前页面没有图片，脚本会给出提示，不会下载。

## 配置说明

脚本顶部有一个最常用的配置项：

```js
const DEFAULT_IMAGE_FORMAT = 'jpg';
```

### `DEFAULT_IMAGE_FORMAT`

支持以下取值：

- `'jpg'`
- `'png'`
- `'original'`

说明：

- `'jpg'` 适合日常保存，兼容性最好
- `'png'` 适合想保留无损格式时使用
- `'original'` 会尽量保留原图格式，可能出现 `webp`

## 适用页面

脚本当前只会在以下页面工作：

- `https://www.xiaohongshu.com/explore/*`
- `https://www.xiaohongshu.com/discovery/item/*`
- 'https://www.xiaohongshu.com/user/profile/*'

## 注意事项

- 这个脚本只针对当前图文笔记做一键保存，不负责批量抓取账号内容。
- 浏览器首次连续下载多张文件时，可能会提示是否允许该站点多次下载文件，需要手动允许。
- 图片格式转换依赖浏览器 Canvas 能力；如果浏览器环境过旧，建议优先使用 `'original'`。
